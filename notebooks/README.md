# Analysis Notebooks

## 02_rfm_analysis.ipynb

Complete customer segmentation analysis combining rule-based RFM with unsupervised K-Means clustering.

**Contents:**
1. RFM calculation and scoring (quintile-based)
2. Business segmentation (10 actionable segments)
3. K-Means clustering for validation (K=4 optimal)
4. Comparative analysis and insights
5. Executive recommendations

**Key Findings:**
- 958 Champions contributing £4.3M (63.8% of revenue)
- 346 At-Risk customers with £548k revenue at stake
- 8 Super-VIP outliers identified (£72k avg. spend)
- 95% validation between RFM and K-Means approaches

**To run:** Requires cleaned data from `01_data_exploration.ipynb`
